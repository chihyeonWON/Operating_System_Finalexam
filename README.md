```
9번. 페이지 기억장치 할당기법에서, 한 페이지의 크기가 512바이트이고 페이지 번호는 0부터 시작한다면,
논리적인 주소 1224번지는 어디로 변환되는가?

0~512 0페이지
512~1024 1페이지
1024~1536 2페이지 
offset = 1224-1024 = 200

2페이지, offset = 200
```
